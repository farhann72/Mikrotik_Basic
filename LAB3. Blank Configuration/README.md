Pada kali ini saya akan menjelaskan tentang blank configuration, "apa itu blank configuartion?"

Blank configuration adalah konfigurasi kosong atau konfigurasi yang tidak ada apa apa di dalamnya, "untuk apa kita belajar??"

Karna blank konfigurasi ini sangat berguna bagi yang mau belajar mikrotik dari 0, karna kita bisa mencoba topologi, mencoba lab, bahkan bisa dijadikan alat praktik khususnya di Networking. Lalu bagaimana cara merubah mikrotik kita menjadi blank configuration

<h1> Cara Merubah Mikrotik Menjadi Blank Configuration</h1>

1. Remove konfigurasi lewat pemeberitahuan

   -Remote mikrotik terlebih dahulu menggunakan WinBox

   -Ingat di awal kita belajar default configuartion? ada dimana pemberitahuan tanda kalau kita sudah masuk default configuration. cara agar kita masuk ke blank configuartion dengan klik Remove Configuartion

   <img width="667" height="483" alt="default config" src="https://github.com/user-attachments/assets/e6ccad2f-aa94-4315-aca1-0cabfdb2be4c" />

   Dengan begitu semua konfigurasi dari default/bawaan sudah terhapus semuanya  dan kita sudah berada di Blank Configuration
